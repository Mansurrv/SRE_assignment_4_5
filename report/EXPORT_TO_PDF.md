# Export report to PDF

The submission requires **a PDF + a Git link**. This repository includes the report in Markdown.

## Option A (pandoc)

If `pandoc` is installed:
```bash
pandoc report/ENDTERM_REPORT.md -o report/ENDTERM_REPORT.pdf
```

## Option B (VS Code / GitHub)

- Open `report/ENDTERM_REPORT.md`
- Use your editor “Export/Print to PDF” feature
